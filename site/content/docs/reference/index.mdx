API reference for the public Streamsy package surfaces. Tables on these pages are
generated directly from the TypeScript source with
[`fumadocs-typescript`](https://www.fumadocs.dev/docs/integrations/typescript), so
they stay in sync with the types the packages actually export.

## How these pages work

Author a table by pointing the `auto-type-table` tag at a source file and an
exported type name:

```mdx
<auto-type-table path="../../../../packages/json/src/json.ts" name="JsonCodec" />
```

- `path` is resolved **relative to the MDX file**, into the monorepo `packages/`.
- `name` is the exported interface / type to document.
- The tag is compiled to a static `TypeTable` at build time (the `remarkAutoTypeTable`
  remark plugin in `source.config.ts`), so no server rendering is needed at request
  time.

Only object-like types (interfaces / object type aliases) produce a table. Use TSDoc
on the source declarations (`@internal`, `@remarks`, `@fumadocsType`, `@fumadocsHref`)
to control what appears.
